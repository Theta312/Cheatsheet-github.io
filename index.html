Position Property

And its 5 Values

The five values

Tags Values Description
<static> Static The default value for borders
<relative> Relative This value lets you position an element relative to its default static position on the web
<absolute> Absolute All other elements will ignore the element and act like its not present on the page. You can change the position with offset properties.
<fixed> Fixed With this Value you set the element fixed in its position on the web page even if you scroll. You control its position with the offset properties
<sticky> Sticky This Value keep an element in the document flow as the user scrolls, but sticks to a specified position as the page is scrolled further. You choose the location to "stick" it to with the offset properties.

Offset attributes

Tag Attribute Description
top Top Moves an element down from the top
left Left Moves an element away from the left side (to the right)
right Right Moves an element away from the right side (to the left)
bottom Bottom Moves an element up from the bottom
